# Kaiser Tech Landing Page

Landing page para a Kaiser Tech, consultoria de tecnologia B2B.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS v4
- i18n local PT-BR/EN

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Notas

- O CTA principal aponta para a seção `Conte sua dor`.
- O formulário usa `mailto:` como placeholder até a escolha do backend/form provider.
- O hero implementa uma animação visual estilo shader em CSS/React. Pode ser substituído pelo componente da 21st.dev quando o código específico estiver disponível.
